STRENGTH

피드백을 좋아합니다. 피드백 주고 받고 이를 통해 스스로 보지 못했던 문제점을 발견하고 개선하는 것을 좋아합니다.

수용력 항상 상대방의 입장을 열린 마음으로 이해하려 노력하고 배려하려는 성격으로 덕분에 원만한 인간관계를 이루어 왔으며, 이러한 성격은 디자이너와 개발자 와의 의사 소통이 중요한 웹퍼블리셔의 장점이 될 수 있다고 생각합니다.

기록과 공유 습득한 지식 정보를 정리하며 기록하는 것을 좋아합니다. 습득한 지식을 다른 이들과 공유하고 나누며 지식의 선순환을 꿈꿉니다.

책임감과 성실함 책임감을 가지고 맡은 일은 끝까지 해내며, 항상 성실한 자세로 최선을 다합니다.

SKILLS

HTML5
웹 표준 및 웹 접근성을 준수하며, 용도에 알맞은 태그를 사용한 시멘틱 마크업 작성이 가능합니다.
또한 타인과의 원활한 의사소통을 위한 간결하고 가독성 좋은 깔끔한 코드 작성을 위해 노력합니다.
CSS3
Bootstrap 등의 css 프레임워크를 활용할 수 있으며 transition과 Animation 을 적용하여 생동감있고 자연스러운 효과를 통해 동적인 웹사이트를 제작하는 것을 즐깁니다.
크로스 브로우징에 대하여 이해하고 있으며, 미디어 쿼리를 이용한 반응형 웹 제작이 가능합니다.
JS / jQuery
자바스크립트의 기본적인 문법을 이해하고 있으며, 주로 제이쿼리와 병행하여 활용합니다.
또한 슬라이드, 모달 창, 탭 메뉴, 네비게이션과 같은 필수적인 요소의 하드코딩이 가능하며, 라이브러리 및 플러그인을 통한 동적 모션 적용도 가능합니다.
PHOTOSHOP
배너, 로고, 이벤트 페이지 그리고 상세 페이지 디자인 제작이 가능합니다. 또한 웹, 모바일 등의 여러 형태의 레이아웃을 구성하며 시안 제작이 가능합니다.
ILLUSTRATOR
간단한 캐릭터나 이모티콘 제작이 가능합니다.
XD
웹 사이트 시안과 프로토타입 제작이 가능합니다.
Git
Git과 Github를 이용한 버전관리가 가능하며 타인과의 원활한 협업을 위하여 공부 중입니다.
React.js
React의 기본 문법을 이해하고 있으며, 향후 프론트엔드 개발자로서 나아가기 위해 공부중입니다.
